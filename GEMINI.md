# GEMINI.md

## プロジェクトについて

- 距離センサー VL53L0X の Python ドライバー
- Raspberry Pi Zero 2W で動かすことを想定
- pigpioを使う。
- numpyなど、高速化に効果的なパッケージは最大限活用する。

## `uv`の利用

- コマンドの実行は、`uv run ...`を使う。
